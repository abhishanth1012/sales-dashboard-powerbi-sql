# Sales Performance Analysis using SQL & Power BI

## Overview
This project analyzes sales data to extract actionable business insights using SQL and Power BI. 
The objective is to understand revenue trends, product performance, and regional sales patterns 
to support data-driven decision-making.

## Tools & Technologies
- SQL (Data querying and analysis)
- Power BI (Dashboard creation and visualization)
- Excel / CSV (Data source)

## Data Workflow
1. Loaded raw sales data into a SQL database
2. Cleaned and transformed data using SQL queries
3. Performed exploratory analysis to identify trends
4. Built an interactive Power BI dashboard with KPIs and filters

## Key Insights
- Identified top-performing products and regions
- Analyzed monthly and yearly sales trends
- Compared category-wise revenue contribution
